# expense-tracker

# 💰 Personal Expense Tracker

A full-stack web application that helps users track daily expenses, manage budgets, and analyze spending patterns through interactive visualizations.

---

## 🚀 Features

### 🔹 User Management

* Secure user authentication
* Password hashing using bcrypt

### 🔹 Expense Management

* Add, edit, and delete expenses
* Categorize transactions
* Track daily spending

### 🔹 Budget Planning

* Set and manage budgets
* Monitor spending vs limits

### 🔹 Data Visualization

* Interactive charts for insights
* Category-wise expense breakdown

---

## 🛠️ Tech Stack

### Frontend

* React (Create React App)
* Axios
* React Router DOM
* Chart.js / react-chartjs-2

### Backend

* Node.js
* Express.js
* MongoDB (Atlas)
* Mongoose
* bcrypt
* CORS
